 🏥 Kohat Medical Center – Appointment Booking Website
📌 Project Overview

This is a frontend practice project for a Medical Appointment Booking Website. It allows users to book appointments by selecting services, doctors, dates, and time slots through a user-friendly interface.

The main goal of this project is to practice HTML, CSS, and JavaScript by building a real-world inspired web application.

🚀 Features

📅 Appointment booking form

👨‍⚕️ Doctor selection

🏥 Multiple medical services

⏰ Time slot selection

📝 Additional notes option

✅ Confirmation modal on submission

📱 Responsive and clean UI design

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout

JavaScript – Form handling and interactivity

Font Awesome – Icons
